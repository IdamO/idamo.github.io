Projects

Alang

Grammar Tracking SmartWatch App

Inspired by fitness tracking apps, Alang helps keep track of and improve grammar by providing corrective feedback to users whenever it encounters a grammatical error in speech. It also tracks data over extended periods of time to show progress.

Languages: C & JavaScript.

Spotify.sh

Voice Activated Spotify Tool

Spoti.sh integrates Bash and AppleScript to enable users control the Spotify desktop client on Mac OS and other UNIX-based operating systems, using the command-line.

Recently utilized Julius open-source to allow voice commands such as: <play artist "Justin Timberlake">.

Languages: Bash, AppleScript & Python.

ClickTweet

Google Chrome Extension

ClickTweet is a simple extension which allows Twitter users to simply right-click the content they want to tweet (webpage, image, link or highlighted text) and tweet directly without having to open Twitter.

Languages: JavaScript, HTML & CSS.

Atari's Breakout

This was my very first project. Breakout is a C-Style implementation of the classic game, Breakout, using Stanford's Portable Library as the GUI.

The challenge arises from the fact that, unlike Java and most other object oriented languages, I had to manually manage memory for the GUI objects.

Languages: C.

TEAM2OARS

Online Apartment Rental Suite

TEAM2OARS is a fully functional apartment rental suite equipped with a database and Django as the framework.

This was a class project for Fundamentals of Software Engineering and it greatly improved my full-stack web development skills.

Languages: HTML, CSS, JavaScript & Python.
